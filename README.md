## Data Description

### Input Data Files (in `data/`)

| File | Description | Format |
|------|-------------|--------|
| `species_relative_abundance.tsv` | Species-level relative abundance profiles generated by MetaPhlAn4 after contaminant removal. Rows = species, Columns = samples. | TSV, taxa-by-sample |
| `taxonomy.tsv` | Taxonomy annotation table with columns: Kingdom, Phylum, Class, Order, Family, Genus, Species. | TSV |
| `metadata.tsv` | Sample metadata | TSV |
| `metabolite_abundance.tsv` | Targeted tryptophan metabolite concentrations measured by UPLC-TQ/MS. Rows = metabolites (e.g., Tryptophan, Kynurenine, Xanthurenate, Indole-3-lactic acid, Nicotinic acid, Nicotinamide), Columns = samples. | TSV, metabolite-by-sample |
| `peptidase_counts.tsv` | Read counts of peptidase genes annotated against MEROPS database from 237 representative MAGs. Rows = peptidase features (e.g., C10.004, S01.283), Columns = samples. | TSV, gene-by-sample |
| `phylogenetic_tree.nwk` | Phylogenetic tree in Newick format used for UniFrac distance calculations. | Newick |

### External Data

- **Longitudinal cohort**: Shotgun metagenomic data from [Saheb Kashaf et al. (2023)](https://doi.org/10.1016/j.chom.2023.03.010), available in NCBI SRA under accession SRP002480. Analysis code is provided in `code/06_longitudinal_validation.R`.
- **Raw sequencing data**: Deposited in the NCBI Sequence Read Archive (SRA) under BioProject ID PRJNA1368396.

## Analysis Overview

### 1. Beta Diversity (`01_beta_diversity.R`)
- Computes weighted and unweighted UniFrac distances
- Performs PCoA ordination
- Runs PERMANOVA to test effects of disease status and anatomical region
- **Reproduces**: Figure 1B, 1C

### 2. Differential Abundance (`02_differential_abundance.R`)
- Runs MaAsLin2 with CPLM model for taxonomic features
- Identifies taxa enriched/depleted in AD vs. Control per region
- **Reproduces**: Figure 2A-C

### 3. Network Analysis (`03_network_analysis.R`)
- Constructs Spearman correlation-based co-occurrence networks
- Computes network statistics (degree centralization, edge density, eigenvector centrality)
- Compares network topology between AD and Control
- **Reproduces**: Figure 3A-C

### 4. Metabolite Analysis (`04_metabolite_analysis.R`)
- Differential abundance of tryptophan metabolites (MaAsLin2, linear model)
- Taxa-metabolite Spearman correlation (focused on xanthurenate)
- **Reproduces**: Figure 4A-D

### 5. Regression and Mediation (`05_regression_mediation.R`)
- Multivariable linear regression linking clinical variables to microbial features
- Causal mediation analysis for parental disease history -> microbiome -> AD
- **Reproduces**: Figure 6A-C

### 6. Longitudinal Validation (`06_longitudinal_validation.R`)
- Linear mixed-effects models with three-way interaction terms
- Tests whether P. pallens / interpain A modulate the S. aureus-SCORAD association
- **Reproduces**: Figure 7B
